# Advocacia Trabalhista - Dr. Bruno de Oliveira

Landing page moderna e responsiva construída em **React** e **Vite** para captação e triagem automatizada de leads qualificados de acidentes de trabalho e doenças ocupacionais no Rio Grande do Sul.

---

## 🛠️ Tecnologias Utilizadas

*   **React 19**
*   **Vite**
*   **Lucide React** (Ícones vetoriais nativos)
*   **CSS3 Vanilla** (Variáveis CSS, flexbox, CSS Grid, Glassmorphism)

---

## ⚙️ Como Executar Localmente

1.  Instale as dependências:
    ```bash
    npm install
    ```

2.  Execute o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

3.  Abra a build de produção:
    ```bash
    npm run build
    npm run preview
    ```
